
 # Somando.js

## 🚀 Sobre o projeto

Somando.js é um projeto simples em JavaScript que realiza operações de soma. O objetivo é praticar lógica de programação e manipulação de números em JavaScript, criando funções fáceis de usar e entender.

## 💡 Funcionalidades

- Soma de dois ou mais números
- Validação de entradas para evitar erros
- Interface simples (se houver frontend)
- Pode ser usado em outros projetos como módulo

## 🛠️ Tecnologias usadas

- JavaScript (ES6+)
- HTML e CSS (se houver interface)
- Git e GitHub para versionamento
